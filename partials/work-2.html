«BUKEE: The Book»

«BUKEE: The Book» - учебный проект на курсе «Продвинутый курс для верстальщиков».

  • HTML5
  • CSS3
  • Adaptive layout
  • БЭМ
  • Flexbox
  • Accessibility
  • Less
  • Gulp
  • SVG
  • Pixel Perfect

Адаптивная сетка, верстка с использованием flexbox, доступность (все интерактивные элементы доступны с клавиатуры), кроссбраузерность (в т. ч. IE11). Выполнена оптимизация изображений, использован SVG-спрайт.
